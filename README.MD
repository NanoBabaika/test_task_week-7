# Комплексные свойства и HTML-редактор

## Выполненные задачи:
1. **Собственный модуль:** Создан модуль `my.complexprop`, расширяющий стандартные типы данных Битрикса.
2. **Интеграция LHE:** Реализовано комплексное свойство с использованием визуального редактора `AddHTMLEditorFrame`.
3. **Универсальность:** Свойство корректно работает как в Инфоблоках (элементы вакансий), так и в Пользовательских полях (разделы инфоблоков).
4. **Решение проблем с LHE:** Реализован механизм перехвата данных из $_POST, решающий проблему "склеивания" имен полей при использовании визуального редактора в динамических комплексных формах.
5. **(Задача со звездочкой):** Реализована поддержка User Fields (UF-полей).

## Инструкция по установке:
1. Загрузить папку `my.complexprop` в `/local/modules/`.
2. Установить модуль через **Marketplace -> Установленные решения**.
3. Для проверки в разделах: создать пользовательское поле с типом `Мое комплексное свойство (UF + HTML)`.
4. Для проверки в элементах: создать свойство инфоблока с типом `Мое комплексное свойство (+HTML)`.
